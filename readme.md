# Music apis

Music apis is a sample for build and secure resful api

### Basic Commands

**Install Dependencies**

```shell
npm install
```

**Start MongoDB**

```shell
mongod
```

**Run Application**

Start server

```shell
npm start
```

Start client

Heroku: https://music-apis.herokuapp.com/

Local: http://localhost:3000/
